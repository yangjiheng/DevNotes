# Development Related Notes

## Ubuntu Development Environment

* Solving -lGL unsupported:  
```bash
sudo apt-get install libgl1-mesa-dev
```
* Solving -lz unsupported: 
```bash
sudo apt-get install zlib1g-dev
```
